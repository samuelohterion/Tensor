# Tensor
Tensor is a **c++ class** that can be used to formulate ***tensor expressions*** with ***Einstein Indices*** in c++ programs as mathematical expressions.  
For example a matrix product:
- in the paper:  
  $$C_{2\times{4}} = A_{2\times{3}} \cdot B_{3\times{4}}$$
  $$A_{j}^i = 1 + 4 i + j$$
  $$B_{j}^i = 1 + 4 j + i$$
  $$C_{j}^i = A_{k}^i B_{j}^k$$
- in your c++ code
  ```
  // 3 Tensors, Matrices 
  Tensor< double >
    A = {2, 3},
    B = {3, 4},
    C = {2, 4};
  
  // 3 Einstein Indices
  EIdx i, j, k;

  A[i][j] =   1 + 3 * i + j;
  B[i][j] = -(1 + 3 * j + i);
  C[i][j] = A[i][k] * B[k][j];

  print(A[i][j], "A");
  print(B[i][j], "B");
  print(C[i][j], "C");
  ```

  out:
  ```
  A
  [ 0..1 ][ 0..2 ]
    1  2  3
    4  5  6

  B
  [ 0..2 ][ 0..3 ]
    -1   -4   -7  -10
    -2   -5   -8  -11
    -3   -6   -9  -12

  C
  [ 0..1 ][ 0..3 ]
    -14   -32   -50   -68
    -32   -77  -122  -167
  ```
